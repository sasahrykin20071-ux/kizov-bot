# Family Bot - Discord бот с веб-интерфейсом

## Концепция и Видение

Discord бот для управления игровой "семьёй" (кланом/группой) с приватным веб-интерфейсом для администраторов. Атмосфера — серьёзная, но дружелюбная. Минимализм с акцентом на функциональность. Управление через браузер, без команд в Discord чате.

## Технологический стек

- **Backend**: Node.js + Express.js
- **Discord**: discord.js v14
- **База данных**: SQLite (через better-sqlite3)
- **Frontend**: Чистый HTML/CSS/JS (без фреймворков)
- **Авторизация**: Сессии с паролем администратора

## Функциональность v1.0 — Система заявок на вступление

### Веб-интерфейс (порт 3000)

**Страница подачи заявки** (`/`)
- Форма: Discord тег (ник#0000), возраст, опыт в играх, причина вступления
- Валидация полей
- Уведомление об успешной подаче

**Панель администратора** (`/admin`)
- Защита паролем
- Список всех заявок с фильтрами (новые/одобренные/отклонённые)
- Кнопки: одобрить / отклонить
- При одобрении — бот отправляет Discord уведомление пользователю

### Discord бот

**Команды (только для админов через веб):**
- Создание канала для общения с заявителем
- Назначение роли семьи
- Отправка личного сообщения заявителю

**События:**
- GUILD_MEMBER_ADD — логирование новых участников

### API Endpoints

| Метод | Endpoint | Описание |
|-------|----------|----------|
| POST | `/api/apply` | Подача заявки |
| GET | `/api/applications` | Список заявок (админ) |
| PUT | `/api/applications/:id/approve` | Одобрить заявку |
| PUT | `/api/applications/:id/reject` | Отклонить заявку |
| POST | `/api/login` | Вход админа |
